# 《漫长的告别》多真相/多结局系统设计

## 核心设计理念

**"真相不是被发现的，是被选择的。"**

同一套线索，在不同的"观测视角"下会指向完全不同的真相。玩家的行为模式——相信谁、怀疑谁、调查什么、轮回多少次——决定了最终看到的"真相"。

这不是"随机分支"，而是**逻辑推导的结果**。每种真相都有独立的、自洽的证据链，玩家看到的真相=玩家自己"选择"相信的那个。

---

## 一、核心变量系统

### 1. 轮回计数器（cycleCount）
- 隐性变量，每次重置自动+1
- 影响：解锁新的对话选项、新的可调查内容、角色的态度变化
- 显示位置：状态面板角落

### 2. 角色好感度（affinity）
```
文穂好感度  [0-100]  初始: 70
灯织好感度  [0-100]  初始: 40
时坂好感度  [0-100]  初始: 0（第4轮回后解锁）
```
- 对话中的选择会影响好感度
- 好感度≥60：角色会透露"第二层"信息
- 好感度≥80：角色会透露"核心秘密"
- 好感度≤20：角色会隐瞒信息甚至误导

### 3. 怀疑度（suspicion）
```
自疑度      [0-100]  初始: 10
疑文穂度    [0-100]  初始: 0
疑灯织度    [0-100]  初始: 5
疑超自然度  [0-100]  初始: 0
```
- **自疑度**：看到与自己有关的反常线索时增加（药物、记忆断层）
- **疑文穂度**：看到文穂的反常行为时增加（秘密行动、隐瞒信息）
- **疑灯织度**：发现灯织与案件有关时增加
- **疑超自然度**：看到无法用常理解释的现象时增加

### 4. 调查方向（investigationTrack）
玩家在游戏中的调查行为会自动累积四个方向的进度：
```
心理方向（psych）    : 调查主角的精神状态、记忆、梦境
犯罪方向（crime）    : 调查现场、指纹、监控、人际关系
超自然方向（occult）  : 调查符号、仪式、古老传说、异常现象
科学方向（science）   : 调查时间规律、锚点物品、物理异常
```

**关键机制**：当某个方向进度≥70%时，该方向的"真相滤镜"激活——所有线索会自动被解读为该方向的解释。

## 二、四种真相详解

### 真相A：「自刃者」—— 凶手是主角自己

**触发条件**（满足任意组合）：
- 轮回次数 ≤ 3 + 自疑度 ≥ 60
- 心理方向进度 ≥ 70%
- 在第2轮回内选择"检查自己的药物"

**核心叙事**：
主角患有多重人格障碍（DID），文穂的"死亡"是主角的"虐绞狂"人格所为。所谓的"轮回"其实是主角在精神崩溃后不断重温同一段创伤记忆——每一次"拯救"都是在试图修正无法接受的现实。

**证据链**：

| 线索 | 表面 | 深层解读 |
|------|------|---------|
| 床头的药瓶 | "感冒药" | 实际上是抗精神病药物（利培酮），标签被撕掉了一半 |
| 主角的记忆闪回 | "预见未来" | 是创伤记忆的碎片重组，不是预知 |
| 文穂死亡现场 | "外部绞杀" | 指纹只有主角一个人的，但主角不记得 |
| 手机里的"庆功宴"照片 | 庆功宴记录 | 经调查，那天根本没有庆功宴，照片是合成的 |
| 电视新闻 | "虐绞狂作案" | 虐绞狂是3年前的案件，早已结案。新闻是主角大脑生成的 |

**叙事逻辑**：
1. 文穂在现实中确实死了（不是轮回，是主角无法接受）
2. 主角是凶手，但"虐绞狂"人格在作案后消失，主人格不记得
3. "轮回"=主角在精神病院/监狱中不断重温创伤
4. 每次"轮回"中主角试图改变结局，但不可能改变已发生的事

**结局分支**：
- **A-1 接受**：主角在 therapy 中接受真相，承认自己的罪行 → 灰暗但"清醒"的结局
- **A-2 否认**：主角拒绝接受，继续轮回 → 永恒的自我囚禁

**主题**：精神疾病的污名化、罪恶感如何摧毁一个人

---

### 真相B：「梦觉」—— 整个故事是寓言式的梦境

**触发条件**：
- 4 ≤ 轮回次数 ≤ 7
- 文穂好感度 ≥ 85 + 灯织好感度 ≤ 20 + 时坂好感度 ≤ 20
- 梦觉方向进度 ≥ 70%（调查场景逻辑矛盾）

**核心叙事**：
整个故事是主角在昏迷/精神崩溃后的梦境。文穂不是一个真实的"他人"——她是主角内心的投射，代表"纯真、温暖、被守护的自己"。所谓"虐绞狂"代表主角内心自我毁灭的冲动。

**证据链**：

| 线索 | 表面 | 深层解读 |
|------|------|---------|
| 星月夜马克杯 | 两人的纪念品 | 杯子上只有一个人的手，另一个"影子"是后来画上去的 |
| 文穂的年龄 | 13岁 | 主角13岁时的记忆投射——那是主角父母去世/被遗弃的年龄 |
| 天气预报 | "下午有暴雨" | 窗外从未下过雨。主角记忆中的"暴雨"是某个创伤事件 |
| 超市监控 | "坏了" | 主角从未真正走出房间。"出门寻找"只是梦中的移动 |
| 时钟停在0点 | 象征死亡时刻 | 0点=起点=终点。梦境的循环结构 |
| 文穂的"等价交换" | 她的口头禅 | 这是主角内心对"公平"的执念——"我付出了什么，就该得到什么" |

**叙事逻辑**：
1. 文穂在现实中不存在（或已经死去很久了）
2. 她是主角在创伤后创造的保护性幻想
3. "杀死文穂"=主角的自我毁灭倾向
4. "拯救文穂"=主角试图拯救内心中那个脆弱的自我
5. "告别文穂"=与创伤和解，接受失去，重新生活

**结局分支**：
- **B-1 告别**：主角选择"放手"，文穂在梦中微笑消失 → 醒来后回到现实世界
- **B-2 沉溺**：主角选择"永远在一起" → 永远困在梦境中（温柔的囚笼）
- **B-3 觉醒**：主角发现文穂就是自己的投射 → 两者融合，成为完整的自我

**主题**：创伤后成长、与过去的自己和解、幻想与现实的边界

---

### 真相C：「深渊」—— 超现实的邪神引发了真实的轮回

**触发条件**：
- 轮回次数 ≥ 8
- 超自然方向进度 ≥ 70%
- 在特定场景（DEATH.gif/black）累计停留超过30秒

**核心叙事**：
主角所在的城市地下封印着一只古老的邪神（或高维存在）。它以"强烈的情感"为食——特别是"爱与失去的爱"。文穂被选为"祭品"，而主角被选中为"饲养员"——不断经历失去，不断产生强烈的情感波动，为邪神提供养分。

**证据链**：

| 线索 | 表面 | 深层解读 |
|------|------|---------|
| 家里的挂钟 | 时间工具 | 钟摆的形状是一个倒五芒星，秒针走动时有微弱的"心跳声" |
| 文穂的草莓发绳 | 装饰品 | 图案是古老的符文——代表"容器"或"祭品" |
| 电视雪花 | "接触不良" | 雪花中隐约组成人脸，每次轮回人脸不同 |
| DEATH.gif背景 | "死亡场景" | 仔细看，背景里有一个巨大的"眼睛"在注视 |
| 时间循环的规律 | "主角的能力" | 不是主角的能力——是邪神在"重置"场景以提取更多情感 |
| 时坂朔的出现 | "偶然遇到" | 他是"守望者"的后裔，世代对抗邪神 |

**叙事逻辑**：
1. 邪神被封印在城市地下的"锚点"中
2. 它需要"纯粹的爱与痛苦"作为能量
3. 文穂的"温暖"+主角的"绝望"=完美的能量组合
4. 每次轮回=邪神"收割"一次情感能量
5. 主角轮回得越多，邪神越强大
6. 最终，当邪神足够强大时，它会破开封印，吞噬整座城市

**结局分支**：
- **C-1 封印**：主角在时坂朔的帮助下，用文穂的"牺牲"重新封印邪神 → 文穂死，城市得救
- **C-2 交易**：主角与邪神交易，以自己的灵魂为代价释放文穂 → 文穂活，主角成为新的"饲养员"
- **C-3 反噬**：主角发现"情感"可以反向流动——用"爱"而非"痛苦"喂养邪神 → 邪神被"净化"，文穂复活（真结局）

**主题**：人类情感的崇高与恐怖、信仰与疯狂的一线之隔

---

### 真相D：「等价交换」—— 诅咒与锚点

**触发条件**：
- 5 ≤ 轮回次数 ≤ 8
- 四个角色好感度相对均衡（没有特别偏向）
- 科学方向进度 ≥ 70% + 调查完成度高

**核心叙事**：
主角的家（bedroom1）是一个天然的时间锚点。主角强烈的不甘与悔恨触发了"轮回"——但这是一种诅咒，不是祝福。每次轮回都在抽取文穂的生命能量作为"燃料"。文穂的家族世代守护锚点，她知道真相，选择配合主角的每一次"拯救"。

**证据链**：

| 线索 | 表面 | 深层解读 |
|------|------|---------|
| 挂钟倒退 | "时间异常" | 每次轮回后倒退1分钟=消耗了1分钟的生命 |
| 草莓发绳褪色 | "旧了" | 颜色越淡=文穂的生命力越少 |
| 文穂的"等价交换" | 她的口头禅 | 这是家族传承的格言——"得到什么，就要付出什么" |
| 死亡时间规律 | "下午5点" | = 主角醒来时间 + 固定时长（燃料耗尽的时间点） |
| 超市购物小票 | "省钱记录" | 文穂在记录"剩余的时间"——每次只买刚好够用的东西 |
| 马克杯上的星月夜 | "纪念品" | 星月夜=梵高的最后一幅画。文穂知道这是"最后的时光" |

**叙事逻辑**：
1. 锚点需要"守护者"来维持稳定
2. 文穂是现任守护者
3. 主角的"轮回"打破了锚点的平衡
4. 不平衡的代价=守护者的生命
5. 文穂知道，但她选择不告诉主角——因为她爱主角
6. 唯一的解法：主角停止轮回，接受失去，让锚点恢复平衡

**结局分支**：
- **D-1 放手**：主角停止轮回，接受文穂的死亡 → 文穂的灵魂融入锚点，成为永恒的守护者
- **D-2 转移**：主角将守护者身份转移到自己身上 → 主角失去实体，以灵魂形式永远守护文穂
- **D-3 共生**：主角发现"情感"本身可以维持锚点——不需要牺牲生命，只需要"真诚的告别" → 两人在不同的存在形式中继续相伴（真结局）

**主题**：爱的代价、放手与守护的悖论、宿命与选择

## 三、隐藏真相E：「观测者」—— 量子叠加态

**触发条件**（最难解锁）：
- 在一周目内解锁全部四种真相的结局
- 二周目时，在关键节点选择"我不相信任何解释"
- 轮回次数 = 13（不吉利的数字，也是文穂的年龄）

**核心叙事**：
四种真相都是"部分正确"。文穂的死亡——以及这个轮回——同时是：
- 一场精神疾病导致的自我毁灭（A）
- 一个创伤后的大脑保护机制（B）
- 一个古老邪神的献祭仪式（C）
- 一个时间锚点的能量失衡（D）

真正的答案是：**四种真相并不互斥，它们是同一个现象的四个观测面。**

就像量子力学中的叠加态——在"观测"之前，粒子同时处于所有可能的状态。文穂的死亡也是如此——在主角"选择相信"之前，它同时是所有可能性的叠加。

**叙事逻辑**：
1. 主角的强烈的情感（爱+绝望）同时触发了四种效应
2. 精神疾病打开了"感知异常"的大门
3. 创伤创造了"需要保护的象征"（文穂）
4. 这个象征恰好与古老的封印产生了共鸣
5. 情感的能量扰乱了锚点的平衡
6. 四种力量交织在一起，创造了一个"既是真实又是幻觉"的叠加态

**唯一结局**：
主角意识到——**不是他在选择真相，是真相在选择他。** 每个轮回中，他的"信念"都在塑造现实。如果他相信A，现实就会变成A；如果他相信B，现实就会变成B。

最终，主角选择**不选择**。他接受了"不确定性"——文穂可能死了也可能没死，这可能是一场梦也可能不是，这可能有邪神也可能没有。

但有一件事是确定的：**他爱文穂。这份爱是真实的，无论真相是什么。**

最后场景：
- 主角站在所有四种结局的"交叉点"
- 文穂的四种形象同时出现——作为受害者、作为幻觉、作为祭品、作为守护者
- 她们同时开口："你想让我成为什么，我就成为什么。"
- 主角微笑："我什么都不想让你成为。做你自己就好。"
- 四种形象融合为一，文穂以"完整的存在"出现——既活着也死了，既真实也虚幻
- 最后一句话："漫长的告别，终于结束了。"

## 四、线索的多重解读系统（核心机制）

### 设计原则

**同一个线索，在不同"真相滤镜"下呈现不同解读。**

玩家收集的线索是**客观的**，但解读方式是**主观的**——取决于玩家的调查方向、好感度、怀疑度。

### 示例：线索「文穂指甲缝里的泥土」

**收集方式**：在第3轮回中调查文穂的手

**四种解读**：

| 真相滤镜 | 解读 | 后续线索 |
|---------|------|---------|
| A-自刃者 | 主角（凶手人格）在埋尸时留下的痕迹 | → 检查院子里的土壤成分 |
| B-梦觉 | 梦境中潜意识的投射——文穂喜欢园艺，但现实中她从不出门 | → 窗外根本没有花园 |
| C-深渊 | 来自"另一个世界"的土壤——邪神领域的一部分 | → 泥土中有不属于这个世界的微生物 |
| D-等价交换 | 文穂死前试图挖掘"锚点的核心"以阻止诅咒 | → 后院有一个被掩盖的挖掘痕迹 |

### 关键设计：线索的"矛盾性"

某些线索在**不同真相滤镜下会互相矛盾**。这正是让玩家"选择相信哪一个"的关键。

例如：
- **监控录像**（如果在某个轮回中修好）
  - A滤镜：监控显示主角自己进入了超市
  - B滤镜：监控根本没有画面——因为超市不存在
  - C滤镜：监控显示文穂在和一个"没有脸的人"说话
  - D滤镜：监控显示文穂在超市买了一些"根本不存在"的物品（可能是锚点相关的道具）

玩家需要决定**相信哪个版本的监控**——而这个选择会影响后续可用的调查方向。

## 五、好感度/怀疑度系统的实现

### 对话选择的影响

每次与角色对话时，选择会影响好感度和怀疑度：

**示例对话（文穂）**：

文穂：「{{user}}，你今天怎么一直盯着我看？」

| 选项 | 好感度影响 | 怀疑度影响 | 解锁线索 |
|------|-----------|-----------|---------|
| "因为你今天特别可爱。" | 文穂+10 | - | 无 |
| "我在想...你会不会有什么事情瞒着我？" | 文穂-5 | 疑文穂+10 | 文穂的微表情变化（后续可调查） |
| "没什么，只是有点累。" | - | 自疑+5 | 无 |
| "我觉得今天的你有点不一样。" | 文穂+5 | 疑文穂+5 | 解锁"文穂的反常日记"线索 |

### 好感度阈值事件

**文穂好感度 ≥ 80**：
- 文穂会主动坦白一些事情：「其实...我有时候会觉得，今天的你已经经历过一次了。」
- 解锁隐藏线索：文穂的抽屉里有一本"记录轮回次数"的手账本

**文穂好感度 ≤ 20**：
- 文穂会开始躲避主角
- 解锁隐藏线索：在文穂的房间发现一封未寄出的信，信中写道「如果他/她发现真相，一切就都结束了」

**自疑度 ≥ 70**：
- 主角开始产生"另一个自己"的幻觉
- 解锁隐藏线索：在镜子里看到"另一个主角"在微笑

## 六、轮回次数的叙事设计

轮回次数不只是"计数"——每次轮回都会改变游戏世界。

### 轮回1-3："正常世界"
- 世界看起来正常
- 只有常规的悬疑线索
- 适合解锁真相A（自刃者）

### 轮回4-6："裂缝世界"
- 开始出现逻辑矛盾（物品位置变化、时间不一致）
- 角色开始"知道太多"
- 适合解锁真相B（梦觉）或D（等价交换）

### 轮回7-10："破碎世界"
- 现实开始崩坏（场景扭曲、NPC说奇怪的话）
- 超自然线索大量出现
- 适合解锁真相C（深渊）

### 轮回11+："观测者世界"
- 所有之前的线索同时存在
- NPC开始打破第四面墙
- 只有在这个状态下才能解锁真相E（观测者）

## 七、系统实现方案

### 需要新增的数据结构

```typescript
// gameStore 新增字段
interface GameState {
  // 现有字段...
  cycleCount: number;           // 轮回次数
  affinity: {
    fumi: number;               // 文穂好感度
    touko: number;              // 灯织好感度
    saku: number;               // 时坂好感度
  };
  suspicion: {
    self: number;               // 自疑度
    fumi: number;               // 疑文穂度
    touko: number;              // 疑灯织度
    occult: number;             // 疑超自然度
  };
  investigation: {
    psych: number;              // 心理方向进度
    crime: number;              // 犯罪方向进度
    occult: number;             // 超自然方向进度
    science: number;            // 科学方向进度
  };
  unlockedClues: string[];      // 已收集线索ID
  truthFilter: 'A' | 'B' | 'C' | 'D' | 'none';  // 当前激活的真相滤镜
  endingsSeen: string[];        // 已看到的结局
}
```

## 八、开场剧本的多真相适配

**核心改动**：开场剧本不再固定揭示"虐绞狂"和"记忆复苏"。相反，开场根据玩家的"第一次选择"来决定初始滤镜。

**玩家第一次行动**：
- 如果选择"先检查自己的身体/房间" → 激活A滤镜（自刃者倾向）
- 如果选择"先关心文穂" → 激活B滤镜（梦觉倾向）
- 如果选择"观察周围的环境异常" → 激活C滤镜（深渊倾向）
- 如果选择"记录时间规律" → 激活D滤镜（等价交换倾向）

**首次死亡后的反应**：
- A滤镜玩家："我...我为什么不记得了？"
- B滤镜玩家："这不像真的...太像一场梦了..."
- C滤镜玩家："有什么东西在看着我..."
- D滤镜玩家："时间...时间在倒流？"

## 九、总结：四条真相线的设计意图

| 真相 | 类型 | 核心体验 | 适合玩家 |
|------|------|---------|---------|
| A-自刃者 | 心理惊悚 | "你以为自己是英雄，结果发现自己是怪物" | 喜欢反转、黑暗叙事的玩家 |
| B-梦觉 | 治愈/哲思 | "告别不是失去，是学会与自己和解" | 喜欢温情、成长主题的玩家 |
| C-深渊 | 超自然恐怖 | "人类的爱与痛苦只是神的饲料" | 喜欢克苏鲁、存在主义恐怖的玩家 |
| D-等价交换 | 宿命浪漫 | "我爱你，所以我必须放手" | 喜欢悲剧美学、宿命论的玩家 |
| E-观测者 | 元叙事 | "真相取决于你相信什么" | 二周目、喜欢哲学思辨的玩家 |

**最终目标**：让每个玩家看到的"真相"都完美契合他们自己的"观测方式"。
